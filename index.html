Objetivo del portafolio

Este portafolio reúne los ejercicios, proyectos y evidencias desarrolladas en la materia de Nuevas Tecnologías. Busca mostrar mi aprendizaje práctico en diseño web, validación de formularios, programación con JavaScript y la construcción de aplicaciones con CRUD básico, además de presentar una organización clara de los recursos y enlaces a cada trabajo.

Elementos aprendidos en Nuevas Tecnologías

  • Creación de páginas web con HTML y estructuración de contenidos.
  • Uso de CSS para dar estilo, orden y diseño a las páginas.
  • Programación de interactividad con JavaScript para validar formularios y controlar la lógica de los ejercicios.
  • Implementación de proyectos básicos de CRUD y rutas en aplicaciones web.
  • Organización de archivos y directorios para mantener un portafolio ordenado.
