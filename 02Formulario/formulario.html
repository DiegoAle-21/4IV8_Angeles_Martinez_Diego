Todos los formularios que estan en internet tienen metodos de programacion por defecto. Estos metodos nos sirven para poder hacer las acciones principales como son: Acciones para poderse conectar con los servidores, a traves de las Arquitecturas Cliete Servidor. Es que vamos a tener por defecto que todos los formularios utlizan en metodo GET, y el segundo metodo principal es POST, aunque existen otros metodos que se encargan del intercabio de información. GET: Este metodo envia por medio de la URL toda la información del formulario a traves del navegador. Su función principal es obtener los datos del servicio; pero es el metodo mas vulnerable de todos, derivado a que se pueden generar inyeccciones de codigo. POST: Este metodo se encarga de registrar una nueva entrada al sistema tambien utiliza la URL como envio por defecto pero su facultad es poder ocultarla, esto significa que los datos no son visibles para el usuario comun en el navegador.
